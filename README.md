## Installation

Clone the Repository:

```
git clone https://github.com/sharif-22/Instagram-UI-Tailwind
```

Navigate to the project

```
cd Instagram-UI-Tailwind
```

Install the Dependencies

```
npm install
```

Run the Project

```
npm run dev
```
